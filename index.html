My Learning Progress: Git & GitHub

Here’s what I’ve learned so far about Git, GitHub, and hosting with GitHub Pages!

What is GitHub?

GitHub is a platform for hosting and sharing code. It lets people collaborate on software projects, track changes, and contribute together.

What is Git?

Git is a version control system that helps you manage changes to files and code over time. It lets you save different versions and work with others without losing progress.

GitHub Desktop

GitHub Desktop is a graphical application that makes it easier to use Git and GitHub without the command line.

How to Create a GitHub Account

  • Go to github.com
  • Click "Sign up" and fill in your details
  • Verify your email address
  • Set your password and profile information

Repository & Commit

  • Repository: A place to store your project files on GitHub.
  • Commit: A snapshot of changes you’ve made to your project. Commits help track your progress and history.

How to Host on GitHub Pages

  1. Create a repository for your website
  2. Add your HTML files (like this one!)
  3. Go to repository settings
  4. Scroll to "GitHub Pages" section
  5. Select a branch and save
  6. Your site will be published at https://AdamsGeeky.github.io/AdamsGeeky/
